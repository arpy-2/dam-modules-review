# dam-modules-review
Material de repaso y recursos de 1º de DAM (Desarrollo de Aplicaciones Multiplataforma). Incluye apuntes, ejercicios y proyectos de los módulos de Lenguajes de Marcas y Sistemas Informáticos. Repositorio diseñado para consolidar los fundamentos técnicos y facilitar el estudio de los conceptos clave del primer año del grado.
